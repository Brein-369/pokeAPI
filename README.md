# pokeAPI

## Install

    $ git clone https://github.com/Brein-369/pokeAPI.git
    $ cd pokeAPI
    $ npm install

## Start & watch

    $ npm run serve

## Simple build for production

    $ npm run build

## Languages & tools

### Vue.JS

- Vuex for data/state management.
- Vuetify for UI/UX and styling

### Axios 

- data fetching.
